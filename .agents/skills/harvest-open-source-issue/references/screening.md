# Issue screening

Score each dimension from 0 to 2 and explain the evidence:

- Requirement clarity
- Maintainer acceptance
- No conflicting assignee or linked PR
- Bounded code surface
- Local testability
- Environment affordability
- Fit with contributor knowledge
- Learning value
- Likely review path

Hard blockers override the numeric score. Examples include an existing active PR, a required design decision that maintainers have not made, unavailable proprietary hardware, or a rejected direction.

Use the result to select one disposition: selected, awaiting-confirmation, research-only, rejected, or blocked.
